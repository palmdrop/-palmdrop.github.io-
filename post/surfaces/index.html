Surfaces - A Recursive Domain Warping Controller

DISPOSITION

  1. Present app
    • “I’ve been developing…”
    • Web interface for recursive domain warping
    • link to previous posts
    • showcase UI, renders, etc
  2. Purpose
    • learn WebGL, GLSL, React
    • build user interfaces
    • fast generation, real time, easy to change settings, colors, etc
  3. Functionality
    • “Divided into three main sections…”
    • Texture
      • Warp effect (amount, iterations, etc)
        • three controllers
      • Noise controllers
        • fractal noise! (link to post)
        • modified noise (link to post)
    • Color
      • general settings
      • controllers for deciding influence of each layer
        • describe mult vs add
    • Render
      • general render settings
      • blue noise for dithering (link to post)
        • problem with webgl1, implementation needs floating point textures
        • other soltuions, like having separate render passes for each layer… didn’t like that, although might try
  4. Usage
    • load app, play around with the sliders.
    • suggested workflow (randomize until you find something, then play with sliders)
    • save image or video
    • feel free to post wherever, but include link
  5. Goals/Aspirations
    • build on this app, solve issues, keep doing web development
    • ideas about more conceptual projects
  6. Final words (call to action)
    • try it out. Post your findings. Link to this post
